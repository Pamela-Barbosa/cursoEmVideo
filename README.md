# 🐍 Desafios de Python - Curso em Vídeo

![Progresso do Curso](https://img.shields.io/badge/Progresso-Mais%20de%20100%20Desafios-brightgreen)

Este repositório foi criado para documentar e armazenar a minha jornada de aprendizado em **Lógica de Programação com Python**, através do renomado curso da plataforma **Curso em Vídeo**. O projeto conta com mais de 100 exercícios práticos que vão desde os conceitos mais básicos até estruturas complexas e boas práticas.

---

## 🎯 Objetivo do Repositório

O principal objetivo deste repositório é consolidar minha base em lógica de desenvolvimento de software, algoritmos e resolução de problemas, servindo também como um portfólio prático da minha evolução na tecnologia.

---

## 🚀 Principais Conceitos Praticados

Durante a resolução dos mais de 100 desafios, os seguintes tópicos foram (e estão sendo) fundamentais:

* **Fundamentos:** Variáveis, tipos de dados primários, operadores aritméticos e manipulação de strings.
* **Estruturas Condicionais:** Tomadas de decisão dinâmicas utilizando `if`, `elif` e `else`.
* **Estruturas de Repetição:** Criação de loops inteligentes com `for` e `while`.
* **Estruturas de Dados:** Organização de dados complexos utilizando **Tuplas**, **Listas** e **Dicionários**.
* **Modularização:** Criação de funções personalizadas, uso de módulos e pacotes para reaproveitamento de código.
* **Tratamento de Erros:** Robustez de software utilizando blocos `try/except` para controle de exceções.
